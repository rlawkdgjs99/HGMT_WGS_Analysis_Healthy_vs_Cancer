Files used for the analysis of each project are as following, which were provided by HGMT Public DB (https://mai.fudan.edu.cn/hgmt/home/): 
  Metadata,
  Relative Abundance of Bacteria,
  Relative Abundance of KOs,

Although various cancer types were analysed, the CRC data were used for my paper, as it is the cancer type of my interest.
Among the CRC project data, only those exhibiting significant difference in Sarcosine metabolism between healthy vs cancer, and significant differential abundance of bacteria were chosen for the paper.
